# cpu_algorithms
This Program Calculates OS Scheduling Algorithms.
You Give Arival Times And Execute Times Of The Processes To The Program, And It Returns Averages Of Turn Around Time And Wait Time.
requirements are:
python
tkinter
pandas
pandastable
matplotlib
